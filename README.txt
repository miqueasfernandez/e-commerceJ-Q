{Spanish}

Este repositorio muestra un proyecto e-commerce enfocado en la compra y venta de productos.
Los usuarios pueden registrarse e iniciar sesión mediante formularios dedicados para acceder a las funcionalidades de compra y venta.
El proyecto está orientado a cualquier tipo de venta de productos.
Está protegido mediante Passport para la autenticación y Bcrypt para el hash de contraseñas, y ha sido desarrollado con Node.js y su framework Express.js.
La aplicación utiliza una base de datos no relacional, en este caso MongoDB, aunque también puede adaptarse para trabajar con SQL.
Además, el proyecto cuenta con un despliegue en Kubernetes y está clusterizado con Docker para un mejor rendimiento del aplicativo.
Se trata de una API RESTful escalable, lista para ser utilizada y modificada según sea necesario.
Actualmente, esta es una versión demo de lo que será una aplicación e-commerce completa.
Si bien aún se encuentra en desarrollo, sus funcionalidades principales ya están implementadas.
La aplicación está desarrollada específicamente en tecnologías backend.
El frontend es únicamente una implementación de prueba realizada con el motor de plantillas Handlebars,
ya que mi especialización se centra exclusivamente en el desarrollo backend.

{English}

This repository showcases an e-commerce project focused on the buying and selling of products.
Users can register and log in through dedicated forms to access the platform's purchasing and selling features.
The project is designed to support any type of product sales.
It is secured using Passport for authentication and Bcrypt for password hashing,
and it is built with Node.js and its associated framework, Express.js.
The application uses a non-relational database—MongoDB—but it can also be adapted to work with SQL. Additionally,
the project is deployed using Kubernetes and containerized with Docker to ensure optimal performance.
It is a scalable RESTful API, ready to be used and extended as needed.
Currently, this is a demo version of what aims to be a fully functional e-commerce application. While the project is still in development,
its core features have already been completed. The application is developed entirely in backend technologies.
The frontend is only a basic test implementation using the Handlebars templating engine, as my focus and expertise lie exclusively in backend development.
